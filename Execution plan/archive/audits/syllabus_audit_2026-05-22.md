## 3. Within-tag importance gradation

**Proposal:** Add a single-character suffix to load-bearing [P]/[ES] tags only:

- **[P:1]** = anchor / read-first / could not cut. The primary on which the module's argumentative arc actually hinges.
- **[P:2]** = strong / second-order — important but a complement to the anchor.
- **[P:3]** = companion / track-conditional — one-among-several; readable but cuttable.

Default: an unmarked [P] = :2. Mark only the :1 anchors (~25–30 items across the programme) and the :3 track-conditional items (~10–15). Apply analogously to [ES:1] for the structurally indispensable secondaries.

## 4. Other dimensions

### [Economist's note] coverage
- **Substantive notes:** Modules 1, 2, 3, 4, 4A, 5, 6, 7, 8, 10, 11B, 11D, 13, 14, 15, 17 — strong.
- **Gaps (where a note would help but doesn't exist):**
  - **Module 3A Lukács:** the reification thesis maps directly onto Hayekian price-as-information and onto behavioural-economic critiques of "rational expectations." No bridge currently.
  - **Module 9 (Social Epistemology):** none. Fricker's hermeneutical injustice has a structural parallel in measurement-theoretic concerns about what concepts can even be operationalised — would help the economist enter the module.
  - **Module 12.B (Pettit, Skinner, Graeber, Scott):** no economist's note. Pettit-republicanism has natural friction with welfare economics; this is a missed bridging opportunity.
  - **Appendix B:** no economist's note at the module level, only embedded in individual entries. The whole appendix *is* about an economist's home territory.

### Phase IV connection flag accuracy
- The [MX] cross-references at the ends of Modules 4, 4A, 6, 7, 8, 9, 11A, 11C, 13, 14, 16 are consistent and accurate.
- **Stale:** the Mead reference in Appendix H.6 ("Module 4A.3") is correct, but Module 18 still has *Note on dissolution* language pointing back to Mead's relocation — the back-pointer is harmless but worth cleaning when next revising.

### Dependency-graph claims
- **Sound:** M1 → M3A → M4 → M5 → M6 → M7 → M8.
