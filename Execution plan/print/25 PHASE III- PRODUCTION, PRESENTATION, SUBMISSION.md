# PHASE III: PRODUCTION, PRESENTATION, SUBMISSION (Months 35–48, 14 months)

**Goal.** A 7,000–10,000 word piece presented at two venues (one informal, one formal), revised, and submitted to a journal (*WestEnd* / *Constellations* / *ZkSP* / *Philosophy & Social Criticism*). Initiate substantive contact with 1–2 scholars from the interlocutor map.

**Study/writing target:** ~330h. **Community engagement:** ~25–30h. **Reading-side heuristic Inv:** ~217h (see [Phase III heuristic Inv](#phase-iii-heuristic-inv-10-bucket-structure) below).

**Phase III Recons framing.** Phase III primarily consumes the Phase I/II Recon corpus. Layer C firing (3-of-5 baseline) adds ~15 frontier Recons in M41–M43, anchored to topic commitment at M35 R5.

### Layer C — R5 conditional-MUST decision (M35)

Five conditional MUST-Anchor promotions, each firing only if Phase III commits to the corresponding track: **7.18 Honneth *Anerkennung* Full** (recognition track), **7.19 *Arbeitende Souverän*** (work / democratic-*Sittlichkeit*), **8.12 Jaeggi *F&R*** (immanent-critique / progress), **7.10 Forst *Noumenal Republic*** (power / justification), **14.9 Aglietta *Régulation*** (capitalism-as-form). Baseline assumption: **3-of-5 firing** (~83h all-in). Decision happens at M35 R5 driven by topic firm-up; distinct from the falling-behind cascade in §A.1.
