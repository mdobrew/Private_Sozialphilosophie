## D. Comprehension infrastructure (detail)

### D.1 Reconstruction-week protocol

Every Full reconstruction week (W5, W14, W21, W36, W44, W48 in Phase I; weekly schedule in Phase II) follows this canonical protocol on top of the inline week instructions:

1. **Practice A reconstruction** (500–700 words): four-part structure — claim / support / directed-against / presuppositions — per [reading_practices.md](reading_practices.md).
2. **Closing sentence** (~1 min): one sentence in the form *"If I were making this argument, the move I would change is X"* — name a premise, method, example, or directed-against opponent. Do not defend yet. Compounds across the programme as a record of your own developing judgments.
3. **H2 component (ii) — Empirical presuppositions** (~10 min): in the reconstruction note, under a subheading `## Empirical presuppositions`, record any empirical claim the author's argument depends on. Practised from M1 onward; pre-loads the full H2 evaluations from M19 onward. (See `tpl-reconstruction.md` — the heading already exists.)
4. **C1 seminar questions** (~20–30 min): file 2–3 questions at the bottom of the reconstruction note — tensions / ambiguities *within the author's own framework*, not things you didn't understand. See Practice C1 in [reading_practices.md](reading_practices.md).
5. **K Use 1 cross-reference at first reconstruction (W5 only):** after your written reconstruction is filed, paste it to Claude with the Claude Use 1 (see reading_practices.md inline callout at Practice A) prompt for a check-against-misreading pass. Hallucination caveat per K Use 4 applies — treat the answer as a hypothesis to verify.

### D.2 Module-close checklist

Every module close (W17, W25, W28, W32, W41, W46, W54 in Phase I; M20, M25, M30, M32, M34 in Phase II; corresponding Phase III boundaries) carries:

| Item | What | Time |
|---|---|---|
| **Comprehension check** | Brief written answers to capability targets listed at module start. | 2–3h |
| **DM (Dialogue Moment)** | State strongest objection to the central argument you just reconstructed; try to answer. With Claude / colleague / in writing. | ~1h |
| **✏ H1 light-discrimination note** | External objection you were tempted to make → translation to internal critique. 150–250 words. **Cadence taper:** every module M1–M6; alternating modules M7–M14 (Phase II); ad-hoc Phase III. | ~30–45 min |
| **C1 tally** | Confirm 2–3 seminar questions filed per major text in the module (already done per §D.1; tally check only). | n/a |
| **✏ Practice C2 — cross-thinker question** | One cross-thinker or translation question. Form 1: *"Does X's framework require what Y calls Z, even when X claims it does not?"* Form 2: *"If we rewrite X's claim in Y's vocabulary, what is lost?"* File alongside G1. Apprenticeship for the M37–38 Comparison Exercise. | ~30 min |
| **🗺 Practice D — Debate map + negative-space note** | Visual map: 2 axes × thinkers covered in the module; place each thinker; mark response arrows. Then 200-word **negative-space note**: name one constructible position no thinker in the module occupies and classify as incoherent / unmotivated / historically suppressed. | ~1.5–2h |
| **📋 QM (Quartals-Meilenstein)** *(W17, W30, M20, M32, M41 only)* | Recent-article comparison — read 1 recent article from *WestEnd* / *DZfPh* / *Constellations*; note what you can follow, can't, and what concepts/thinkers you're missing. Replaces that week's bonus reading. | ~2–3h |

**Module-close gross load:** ~5–7h non-QM weeks, ~7–10h on QM weeks.
