## B. Session-structure key

| Hint | Meaning |
|------|---------|
| `[dense — 1h daily, don't skip >1 day]` | Hegel *Rechtsphilosophie*, Hegel *Phänomenologie*, Marx *Kapital* ch.1, Lukács reification essay, Adorno *DdA* + *Minima Moralia*, Habermas *EuI*. If a day is missed, re-read the previous 2 pages. |
| `[dense — 1h daily; one 2h sitting on the following passage]` | Used where a long argument arc (50pp+) rewards consolidation. Specified inline. |
| `[systematic — 1h daily; §§X–Y in one 2h sitting if possible]` | Used for Neuhouser, Habermas TKH (in Phase II), and dense secondaries. |
| `[accessible — 1h daily]` | Weber, Dewey, Fricker, Heinrich, McCarthy, most journal articles. No deviation. |
| `[writing — 2h+ block, not 1h]` | Practice tasks, comprehension checks, field reports, transition retreats. |
