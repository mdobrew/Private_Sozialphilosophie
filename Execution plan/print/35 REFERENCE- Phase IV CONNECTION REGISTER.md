# REFERENCE: Phase IV CONNECTION REGISTER

Two views of the same information. The **by-track** view (below) lists each Phase IV deepening track and where in Phase I–III it connects — useful when deciding which Phase IV reading to chase from a given Phase I/II encounter. The **by-week** view further down maps Phase I/II/III weeks to specific Phase IV items they generate — useful when scheduling, and as the working register to update as you encounter [MX] / Appendix H flags during reading.
